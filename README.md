# Personal CV — Mochammad Henry Alifian

A single-page personal CV website built with **HTML5** and **CSS3** for the Web Programming (PWeb) course at Institut Teknologi Sepuluh Nopember.

## Preview

Open `index.html` directly in a browser, or serve locally:

```bash
npx -y http-server -p 8080 -o
```
